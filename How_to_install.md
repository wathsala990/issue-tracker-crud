### Installing

- folders

- - client : all frontend files
- - server : entire backend database


## Install NPM

1. goto client via cmd, install `npm istall`
2. goto server via cmd, install `npm istall`

## Start Project

1. for client - `npm run dev`
2. for server - `npm start`

### Databse (IMPORANT)

- `role.json` file in `server/db` folder
- import that file to mongodb compass (before this project must correctly run `backend`)
